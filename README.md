# Apk-Pengelolaan-Data-Sampah-dan-Daur-Ulang
apk ini di rancang berbasis Comand Line Interface (CLI) dengan menggunakan bahsa pemrograman GO (Golang) guna dalam pengelolaan catatan data sampah dan proses daur ulangnya dalam suatu wilayah. Aplikasi ini di rancang agar dapat digunakan oleh petugas kebersihan maupun masyarakat umum dalam riwayat pencatatan data sampah secara terstruktur.
